#  Kegelbuch

Digitales Kegelbuch zur Verwaltung von Kegelabenden, Spielern und Ergebnissen.

## Schnellstart

```bash
npm install    # Dependencies installieren
npm run dev    # Entwicklungsserver starten (http://localhost:5173)
```
